Использование фильтров
======================

Фильтры являются одной из самых сильных особенностей Magnetto. Они позволяют
уточнить процедуру проведения поиска, процедуру обработки полученных результатов.

По умолчанию во всех методах ``BaseApi`` наследуются Фильтры
``OrderBy.DOWNLOADS`` и ``Order.DESC``, которые отвечают за сортировку по
количеству скачиваний и сортировку по убыванию соответственно. В случае если
переданы фильтры отличные от типа ``OrderBy`` и ``Order``, то данные фильтры
добавляются к фильтрам, переданным в функцию.

.. warning::

  Методы ``BaseApi`` поддерживают только 1 фильтр одного типа. В случае
  необходимости передачи нескольких фильтров, например: ``Category.FILMS`` и
  ``Category.GAMES`` необходимо несколько раз вызвать требуемый метод указывая
  сначала ``Category.FILMS``, а затем ``Category.GAMES``
